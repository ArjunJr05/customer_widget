# Changelog

## [1.0.0] - 2025-11-30

### Added
- Initial release of SalesIQ Customer Widget package
- Complete customer data tracking and display
- Order management and tracking
- Support issue reporting and tracking
- Customer analytics and insights
- Automatic SalesIQ integration
- Local data storage with SharedPreferences
- Webhook server integration
- Development and production configurations
- Comprehensive example app
- Full documentation and setup guide

### Features
- **Customer Profile Management**: Track customer information, loyalty status, and analytics
- **Order Tracking**: Complete order lifecycle tracking with status updates
- **Support Integration**: Issue reporting and resolution tracking
- **Real-time Sync**: Automatic data synchronization with webhook server
- **SalesIQ Integration**: Seamless integration with Zoho SalesIQ chat widget
- **Startup Focused**: Simple, cost-effective solution for growing businesses

### Supported Platforms
- Android
- iOS
- Web (limited SalesIQ support)
